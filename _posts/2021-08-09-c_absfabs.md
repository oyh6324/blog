## abs()

정수, int 타입의 절댓값을 반환한다. <stdlib.h>이 필요하다.

## fabs()

double 타입의 수의 절댓값을 반환한다. <math.h>이 필요하다.

## 사용법

```c
#include<stdlib.h>
#include<math.h>

int main()
{
	int iNum = -20;
	double dNum = -5.1;

	printf("%d, %f", abs(iNum), fabs(dNum));
	return 0;
}
```

이렇게 하면 20과 5.1이 출력될 것이다!
