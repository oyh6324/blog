## <limits.h>란?

여러 자료형의 최소값이나 최대값, 바이트 수를 알 수 있는 매크로가 담긴 헤더 파일이다. C언어의 표준 라이브러리임!

## 매크로

![image-center]({{ site.url }}{{ site.baseurl }}/assets/images/c/c.limits.jpg){: .align-center}
_limits.h에 담긴 매크로_

위키백과에서 들고 옴!

## 사용법

```c
#include <stdio.h>
#include <limits.h>

int main()
{
int  maxNum = INT_MAX;
int minNum = INT_MIN;

return 0;
}
```

이렇게 하면 max에는 2147483647가 들어가고, min에는 -2147483648이 들어갈 것이다!
