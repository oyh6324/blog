## 이중 포인터란?

포인터 변수의 주소를 가지는 포인터를 이중 포인터라고 한다. `이중 포인터 → 포인터 → 변수` 이런 느낌?!

## 사용법

```c
#include <stdio.h>

int main() {
	int a = 5;
	int *ptr = &a;
	int **ptr = &ptr;

	return 0;
}
```

이중 포인터는 **를 사용하며, 위 코드대로 하면 **ptr은 5가 될 것이다! 포인터는 *를 사용할수록 삼중, 사중으로 만들 수 있다!
