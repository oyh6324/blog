## 🌟 C++의 진입점(Entry Point)

```c
#include <iostream>
usnig namespace std;

int main()
{
    return 0;
}
```

엔트리 포인트란 진입점을 뜻한다. C++은 C와 동일하다. int main을 사용! 



그리고 iostream을 사용하며 `usnig namespace std`을 해서 나중에 cout과 cin을 편하게 쓸 수 있다. cout과 cin은 가장 편한 입출력 함수!
