## 🌟 HashMap이란?

전에 봤던 해쉬테이블과 비슷한 자료구조다. 키와 값을 저장하며, 저장되는 순서나 위치는 관계가 없고 프로그래머는 알 수 없다.



## 🌟 사용법

```java
HashMap<Integer, String> hm = new HashMap<>();
hm.put(1, "a");
hm.remove(1);
...
```
