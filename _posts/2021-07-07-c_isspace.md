## isspace()

`isspace()`는 bool 값을 반환하는 공백 확인 함수다. 

## 사용법

```c
int main() {
	char c;
	while ((c = getchar()) != '.')
	{
		if (isspace(c))
			printf("공백을 입력했습니다.");
	}
}
```

`getchar()`로 c에 문자를 입력받고 `isspace()`로 공백인지 아닌지 확인하는 코드다. 공백이면 true를 반환해서 조건문 안의 `printf()`가 출력될 것이다. 참고로 `while ((c = getchar()) != '.')` 이 부분은 '.'이 나오기 전까지 계속 문자를 입력받겠다는 의미!
