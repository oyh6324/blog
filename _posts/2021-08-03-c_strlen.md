## strlen()

null을 제외한 문자열의 길이를 반환하는 함수다.

## 사용법

```c
#include <string.h> //string.h가 필요함!

int main()
{
	char str[10] = "abc";
	int len = strlen(str);
}
```

정수형 변수 len에 str의 길이가 들어갈 것이다. 영어는 하나당 1 바이트이므로 len은 3!
