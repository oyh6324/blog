## 🌟 표준 입출력

C++에서는 보통 입출력할 때 cin과 cout을 사용한다.



기본 형태는 

```c
std:: cout << "Hello";
```

이렇게 생겼지만, 앞에서 설명했듯이 std라는 namespace를 사용하면 `std::` 이 부분을 생략할 수 있다.



cin도 마찬가지!



## 🌟 사용법

```c
#include <iostream>
using namespace std;

int main()
{
    int a;
    
    cin >> a;
    cout << a << endl;
    
    return 0;
}
```

cin은 cout과 반대로 `>>`를 사용해야 하고, 데이터 입력을 받을 수 있다. cout의 `endl`은 줄바꿈을 해주는 역할이다.
